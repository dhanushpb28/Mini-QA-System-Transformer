# Mini-QA-System-Transformer
The Mini-QA-System-Transformer is a simple yet powerful Question-Answering (QA) system built using a pretrained Transformer model from Hugging Face. It is designed to answer user questions based on a given context paragraph from a specific domain Health  

Topic: Mental Health

Example:

``Ask a question about mental health: What helps reduce stigma around mental health?``
``Answer: Mental health education in schools and workplaces.``
``Confidence Score: 0.87``
